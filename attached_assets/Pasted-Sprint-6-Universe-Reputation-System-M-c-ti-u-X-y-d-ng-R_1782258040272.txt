Sprint 6 — Universe Reputation System

Mục tiêu

Xây dựng Reputation System trung tâm cho toàn bộ Universe Ecosystem.

Đây KHÔNG phải Marketplace Reputation.

Đây là Universe Reputation dùng chung cho:

- Universe Hub
- Football Universe
- Animal Evolution
- World Creator
- SafePass
- Exchange Hub
- Các app tương lai

Tất cả app có thể cộng/trừ reputation thông qua Universe Account API.

Database Schema

Tạo file:

lib/db/src/schema/reputations.ts

Schema:

reputations
(
 id uuid primary key default gen_random_uuid(),

 user_id uuid not null unique,

 score integer not null default 0,

 level text not null default 'CITIZEN',

 positive_events integer not null default 0,

 negative_events integer not null default 0,

 last_activity_at timestamp,

 created_at timestamp default now(),

 updated_at timestamp default now()
)

──────────────────────────────

)

Business Rules

Nếu reputation chưa tồn tại:

auto create

──────────────────────────────

addReputation()

score += points

positiveEvents += 1

lastActivityAt = now()

──────────────────────────────

deductReputation()

score -= points

negativeEvents += 1

lastActivityAt = now()

──────────────────────────────

Luôn lưu event:

{
 sourceApp,
 reason,
 points
}

vào reputation_events

──────────────────────────────

Level Up Detection

Nếu level thay đổi:

CITIZEN → TRUSTED

TRUSTED → ELITE

ELITE → LEGEND

LEGEND → MYTHIC

thì tự động gửi Notification.

Sử dụng NotificationService của Sprint 5.

Notification:

title:
🎉 Thăng cấp danh tiếng

message:
Bạn đã đạt cấp độ TRUSTED.

type:
ACCOUNT

priority:
HIGH

sourceApp:
universe-account
