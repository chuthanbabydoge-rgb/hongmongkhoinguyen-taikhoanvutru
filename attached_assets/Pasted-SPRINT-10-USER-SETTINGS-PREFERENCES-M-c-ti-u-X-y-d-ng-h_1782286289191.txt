Tích hợp Activity Feed (Sprint 7).

Inject ActivityService vào UserSettingsService.

Khi update settings:

record activity:

type = ACCOUNT

title = Settings Updated

sourceApp = universe-account

visibility = PRIVATE

Khi reset settings:

record activity:

type = ACCOUNT

title = Settings Reset

sourceApp = universe-account

visibility = PRIVATE

Tích hợp Notification Center (Sprint 5).

Inject NotificationService.

Khi thay đổi:

privacyProfile

privacyActivity

privacyReputation

=> gửi ACCOUNT notification priority NORMAL.

Khi tắt:

securityNotifications

=> gửi SECURITY notification priority HIGH.
