Sprint 7 — Activity Feed (Universe Timeline)

Mục tiêu

Xây dựng Activity Feed trung tâm cho toàn bộ Universe Ecosystem.

Đây là Timeline dùng chung cho:

- Universe Account
- Universe Hub
- Football Universe
- Animal Evolution
- World Creator
- SafePass
- Exchange Hub

Mọi hệ thống đều có thể ghi hoạt động vào Activity Feed.

Activity Feed sẽ là nguồn dữ liệu chính cho:

- Universe Hub Dashboard
- User Profile Timeline
- Notification Center mở rộng
- Social Layer tương lai

Database Schema

Tạo file:

lib/db/src/schema/activities.ts

Schema:

activities
(
 id uuid primary key default gen_random_uuid(),

 user_id uuid not null,

 type text not null,

 source_app text not null,

 title text not null,

 description text,

 metadata jsonb,

 visibility text not null default 'PUBLIC',

 created_at timestamp default now()
)
